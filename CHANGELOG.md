# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-10-27
### Añadido
- Implementación completa del motor de orquestación de agentes.
- Módulo de Paper Trading con soporte para NVDA, AAPL y MSFT.
- Dashboard con estética 'Premium Fintech' usando shadcn/ui.
- Suite de pruebas unitarias e integración con Jest.
- Integración de Prisma con soporte para SQLite.

## [0.2.0] - 2023-10-25
### Añadido
- Diseño de arquitectura técnica detallada.
- Definición de stack tecnológico (Next.js, FastAPI, Prisma).

## [0.1.0] - 2023-10-24
### Añadido
- Documento de Requerimientos del Proyecto (PRD).
